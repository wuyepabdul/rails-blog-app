# Rails Blog App

> This project is a blog application built with Ruby on Rails

## Built With

- Ruby
- Ruby on Rails

## Getting Started

```
git clone https://github.com/wuyepabdul/rails-blog-app
cd rails-blog-app
bundle install
rails s
```

## Tests
```
git clone https://github.com/wuyepabdul/rails-blog-app
cd rails-blog-app
bundle install
gem install rspec
rspec
```
